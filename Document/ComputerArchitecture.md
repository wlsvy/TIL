##### 폰 노이만 아키텍쳐 (von neumann Architecture)
ENIAC 개발 프로젝트의 고문 존 폰 노이만John von Neumann이 제안한 `프로그램 내장식 컴퓨터 stored program computer` 입니다. 컴퓨터 내부에 프로그램과 데이터를 저장하여 컴퓨터가 필요한 내용을 순서에 따라 인출하고 해독하도록 구성하였습니다.
